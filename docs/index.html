📡 Raspberry Pi SDR Station

Receiving and decoding real-world radio signals — from FM broadcasts to the ISS in orbit.

About this project

A software-defined radio system built on a Raspberry Pi 5 using a RTL-SDR dongle. The goal is to receive and decode five different types of radio signals entirely in software, covering core ECE concepts including IQ sampling, FFT, digital demodulation, and orbital mechanics.

Built independently as an engineering project. View source on GitHub →
